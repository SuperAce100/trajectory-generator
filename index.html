Robot Path Planning

Controls

  • Arrow Keys: Move robot manually
  • A/D: Rotate robot
  • Space or Button: Go to target

Path Planning

  • Click: Set target position
  • Second Click: Set target orientation
  • Shift + Click: Add obstacle
  • O: Toggle obstacle visualization

About

This simulation demonstrates autonomous robot path planning with obstacle avoidance. The robot uses a gradient descent algorithm to find optimal paths while considering kinematic constraints and obstacles.
